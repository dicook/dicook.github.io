1. XLispStat needs to be installed on your computer. For information on XLispStat click here . This is the way that I have it setup:
  2. Add a line to your .mime.types file (maybe the system administrator needs to do this.)

    application/x-xlispstat xli

  3. Add a line to your personal .mailcap file:

    application/x-xlispstat; $HOME/run_xlispstat %s

    where run_xlispstat is an executable shell script that has two lines: 

       cd /usr/local/xlispstat/xlispstat-3-44
   xlispstat="xterm -e xlispstat"
Some web browsers allow you to set this up interactively, for example, in netscape using the "Options" and "General Preferences" leads you through the setup.
